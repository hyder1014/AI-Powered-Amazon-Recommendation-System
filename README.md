#AI-POWERED-AMAZON RECOMMENDATION SYSTEM
## Overview
This project is a machine learning-based Amazon product recommendation and search engine system. It helps users search for products and receive relevant recommendations using NLP and similarity techniques.

## Features
- Product search functionality
- Recommendation system
- NLP-based text processing
- Similar product suggestions
- Dataset analysis using Jupyter Notebook

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLP
- Jupyter Notebook

## Project Structure
```bash
├── app.py
├── amazon_product.csv
├── README.md
├── requirements.txt
├── notebook.ipynb
└── img/
```

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/Search-Engine-And-Recommendation-System-on-Amazon-Product.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python app.py
```
